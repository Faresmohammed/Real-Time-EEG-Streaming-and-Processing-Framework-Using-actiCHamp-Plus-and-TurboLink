Real-Time EEG Streaming and Processing Framework Using actiCHamp Plus and TurboLink

This MATLAB-based system streams real-time EEG data from the actiCHamp Plus amplifier via TurboLink to a client computer using UDP communication. The program decodes EEG packets, validates stream integrity, visualizes live EEG signals, and performs real-time spectral analysis for online brain signal processing.

How to Run

Connect the EEG amplifier and start data streaming in TurboLink.
Configure the UDP port and sampling settings in the MATLAB script.
Run the script in MATLAB to start real-time EEG acquisition, live visualization, and PSD analysis.
